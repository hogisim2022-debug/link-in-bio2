# Starbucks Website — UI Kit

A high-fidelity recreation of the Starbucks marketing/ordering website.

## Screens demonstrated (single-page)
- **Top nav** + top-of-page Join strip
- **Split hero** — cream canvas, product photo, primary CTA
- **House Green feature band** — "Free coffee is just the beginning" Rewards pitch
- **Rewards tier cards** — 3-column Welcome / Green / Gold panels
- **Gift cards grid** — illustrated tiles grouped by occasion
- **Dark footer** — House Green with gold/white links
- **Frap** floating order CTA

## Components
- `components.jsx` — `TopJoinStrip`, `GlobalNav`, `Hero`, `FeatureBand`, `RewardsTiers`, `GiftGrid`, `Footer`, `FrapButton`, `Section`
