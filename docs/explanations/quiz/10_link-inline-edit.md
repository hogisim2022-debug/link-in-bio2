# 복습 퀴즈 — 10. 링크 수정 인라인 에디터

**연계 문서:** `09` 링크 추가 폼, `10` 링크 수정 인라인 에디터

---

1. 인라인 편집 중에는 `draft` state만 바뀌고 실제 `links` state는 그대로다.
2. 여러 링크를 동시에 인라인 편집 모드로 열 수 있다.
3. `border-top: none`으로 기본 행과 인라인 폼이 하나처럼 붙어 보이게 했다.
4. Supabase 연동 후 저장 실패 시 `cancelEdit`로 롤백한다.

---

## 정답

1. O — 저장 버튼을 눌러야 실제 `links` state에 반영된다.
2. X — 한 번에 하나의 링크만 편집 모드로 열린다. `editingId`로 어떤 링크가 편집 중인지 추적한다.
3. O — 기본 행과 인라인 폼이 시각적으로 연결되어 보이도록 처리했다.
4. O — DB UPDATE 실패 시 `draft`를 원래 값으로 되돌린다.
