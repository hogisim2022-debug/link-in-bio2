# 복습 퀴즈 — 06. Lucide React 아이콘

**연계 문서:** `00` 디자인 토큰, `06` Lucide 아이콘

---

1. SVG는 픽셀 기반 이미지라 확대하면 선이 깨진다.
2. `lucide-react`의 아이콘은 CSS로 색상과 크기를 조절할 수 있다.
3. `GripVertical` 아이콘은 드래그 핸들로, `LogOut`은 로그아웃 버튼으로 사용된다.
4. 이모지를 아이콘으로 쓰면 모든 기기에서 동일하게 보인다.

---

## 정답

1. X — SVG는 수학적 좌표와 선으로 그리기 때문에 확대해도 깨지지 않는다.
2. O — `color`, `size` props 또는 CSS로 스타일을 조절할 수 있다.
3. O — 이 프로젝트에서 실제로 사용된 아이콘들이다.
4. X — 이모지는 OS마다 다르게 보인다. iOS, Android, Windows에서 모양이 다르다.
