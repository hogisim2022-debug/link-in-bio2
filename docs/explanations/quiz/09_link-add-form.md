# 복습 퀴즈 — 09. 링크 추가 폼

**연계 문서:** `08` Link Manager, `09` 링크 추가 폼

---

1. 링크 추가 시 `order` 값은 현재 목록 길이 + 1이 된다.
2. Supabase 연동 후에도 임시 ID(`Date.now()`)를 계속 사용한다.
3. 추가 버튼을 누르면 폼이 자동으로 닫히고 목록에 새 링크가 나타난다.
4. 새로 추가된 링크의 `click_count` 초기값은 0이다.

---

## 정답

1. O — 목록 맨 끝에 추가되도록 `links.length + 1`로 설정한다.
2. X — Supabase 연동 후에는 DB가 생성하는 uuid로 교체한다.
3. O — 추가 후 폼을 초기화하고 닫는다.
4. O — 새 링크는 클릭 수 0에서 시작한다.
