# 복습 퀴즈 — 디자인 시스템 토큰

**연계 문서:** `00_0-4_design-system-tokens.md`

---

1. CSS 변수 이름은 반드시 `--`로 시작해야 한다.
2. 이 프로젝트에서 `--sb-` 접두사의 `sb`는 Supabase를 가리킨다.
3. `62.5%` 폰트 설정을 쓰는 이유 중 하나는 rem 값을 px로 바로 계산하기 쉽기 때문이다.
4. CSS 변수는 JavaScript에서도 런타임에 값을 변경할 수 있다.
5. 여러 컴포넌트 파일에서 같은 색상을 쓰더라도, 디자인 토큰을 쓰면 한 곳에서만 수정해도 전체에 반영된다.

---

## 정답

1. O — `--이름: 값` 형식이 CSS 변수 선언 규칙이다.
2. X — Starbucks의 약자다. 이 프로젝트의 디자인 시스템이 Starbucks 디자인 시스템에서 영감을 받았기 때문이다.
3. O — `1rem = 10px`이 되어 `2.4rem = 24px`, `3.2rem = 32px` 처럼 10의 배수로 바로 계산된다.
4. O — 다크모드나 테마 변경 등에 활용할 수 있다.
5. O — `:root`의 변수 값 하나만 바꾸면 그 변수를 참조하는 모든 곳에 즉시 반영된다.
