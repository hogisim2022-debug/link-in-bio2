# 복습 퀴즈 — 12. SEO Settings 폼

**연계 문서:** `03` Public 페이지 포팅, `12` SEO 폼

---

1. `og:image`를 설정하지 않으면 카카오톡 공유 시 이미지가 표시되지 않는다.
2. Meta Description 권장 글자 수인 160자를 넘어도 검색엔진 노출에는 영향이 없다.
3. `seo.title.length`는 별도 계산 함수 없이 state에서 직접 읽어 실시간으로 갱신된다.
4. `<textarea>`에 `rows={3}`을 설정하면 사용자가 더 크게 늘릴 수 없다.

---

## 정답

1. O — OG 이미지가 없으면 공유 카드에 이미지 없이 텍스트만 표시된다.
2. X — 160자를 넘으면 검색 결과에서 잘려서 표시되어 클릭률에 영향을 준다.
3. O — state가 바뀔 때마다 React가 `<p>`를 다시 렌더링하므로 실시간으로 숫자가 갱신된다.
4. X — `rows={3}`은 초기 높이다. `resize: vertical` CSS로 사용자가 수동으로 늘릴 수 있다.
