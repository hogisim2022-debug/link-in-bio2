# 복습 퀴즈 — 14. 데이터베이스 기초 & CRUD

**연계 문서:** `14` DB 기초, `15` Supabase DB & RLS

---

1. 엑셀의 "시트"는 데이터베이스의 "테이블"과 비슷한 개념이다.
2. 방문자가 공개 페이지에서 링크 목록을 보는 동작은 CRUD 중 Create(C)에 해당한다.
3. `user_id` 컬럼은 데이터의 소유자가 누구인지 표시하는 데 사용된다.
4. 데이터베이스는 엑셀과 달리 여러 사람이 동시에 데이터를 읽고 쓸 수 있다.
5. 링크 제목을 수정하는 것은 CRUD 중 Update(U)에 해당한다.

---

## 정답

1. O — 엑셀 파일 = 프로젝트, 시트 = 테이블, 행 = 레코드로 대응된다.
2. X — 목록을 보는 것은 Read(R)다. Create는 새 데이터를 추가하는 동작이다.
3. O — `user_id`가 로그인 계정 ID와 일치하면 "내 데이터"로 구분한다.
4. O — 데이터베이스는 수천 명이 동시에 접속해도 안전하게 처리한다.
5. O — 기존 데이터를 수정하는 동작이 Update(U)다.
