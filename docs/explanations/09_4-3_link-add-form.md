# 링크 추가 폼

**완료한 task:** `4. Admin 페이지 > 4-3. Link Manager > 링크 추가 폼`  
**수정 파일:** `components/admin/LinkEditor.tsx`, `app/globals.css`

---

## 이 문서를 읽고 나면 풀 수 있어요

1. 폼 입력으로 링크를 추가하면 즉시 DB에 저장된다.
2. Supabase 연동 전 임시 ID로 현재 시각(밀리초)을 쓰는 이유는 중복 가능성이 매우 낮기 때문이다.
3. 새 링크를 추가할 때 기본 공개 여부(`is_visible`)는 false다.

---

## 무엇을 했나?

링크 목록 아래 "+ 링크 추가" 버튼을 누르면 폼이 펼쳐진다.  
제목, URL, 타입을 입력하고 추가하면 목록과 Live Preview에 즉시 반영된다.

**버튼 상태 (폼 닫힘)**
```
┌ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┐
          +  링크 추가
└ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┘
```

**폼 열린 상태**
```
┌──────────────────────────────────────┐
│  제목                                │
│  ┌──────────────────────────────┐    │
│  │  카카오톡 문의               │    │
│  └──────────────────────────────┘    │
│  URL                                 │
│  ┌──────────────────────────────┐    │
│  │  https://                   │    │
│  └──────────────────────────────┘    │
│  타입                                │
│  ┌──────────────────────────────┐    │
│  │  기본형                  ▼   │    │
│  └──────────────────────────────┘    │
│  ┌──────────────────┐  [ ✕ ]        │
│  │       추가       │               │
│  └──────────────────┘               │
└──────────────────────────────────────┘
```

## 핵심 개념 1: 새 링크 객체 생성

추가 버튼을 누르면 폼 입력값으로 새 링크를 만들어 목록 끝에 붙인다.

| 필드 | 값 |
|------|----|
| id | 현재 시각(밀리초) — Supabase 연동 후 uuid로 교체 |
| title / url / type | 폼 입력값 |
| order | 현재 목록 길이 + 1 (맨 끝 순서) |
| is_visible | true (기본 공개) |
| click_count | 0 |

- ❌ 과거: `"새 링크 추가할 때 ID 어떻게 만들어?"`
- ✅ 현재: `"Supabase 연동 전이니까 id: \`link-${Date.now()}\` 형식으로 임시 ID 써줘. 연동 후 uuid 교체할 TODO 주석 남겨줘"`

## 퀴즈 정답

1. 폼 입력으로 링크를 추가하면 즉시 DB에 저장된다. → **X**  
   ↳ 현재는 로컬 상태에만 추가된다. Supabase 연동 후 DB INSERT가 추가된다.

2. Supabase 연동 전 임시 ID로 현재 시각(밀리초)을 쓰는 이유는 중복 가능성이 매우 낮기 때문이다. → **O**  
   ↳ 밀리초 단위라 같은 값이 나올 확률이 매우 낮다.

3. 새 링크를 추가할 때 기본 공개 여부(`is_visible`)는 false다. → **X**  
   ↳ 기본값은 `true`(공개)다.
