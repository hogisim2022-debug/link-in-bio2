# Starbucks Mobile App — UI Kit

A high-fidelity recreation of the Starbucks mobile app, based on the Korean Starbucks reference screenshot (`assets/reference/starbucks-korea-app.png`).

## Screens demonstrated
- **Home** — greeting, stars progress, What's New / Coupon / notifications, recommended menu, Delivers CTA
- **Rewards** — My Rewards with progress, tier detail (Welcome / Green / Gold), benefits list
- **Order** — basic order entry landing

## Components
- `components.jsx` — shared: `PhoneFrame`, `StatusBar`, `TabBar`, `PillButton`, `StarRow`, `MenuCard`, `TierBenefit`

## How to run
Open `index.html` — use the tab bar at the bottom to switch screens.

## Visual source
Cross-referenced the uploaded reference screenshot for: layout proportions, tab-bar icon set, progress-bar styling, membership tier chips, Rewards history pill, and body copy localization.
